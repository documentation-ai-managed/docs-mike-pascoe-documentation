## API Integration

For developers, Linktree offers a robust API.

| Endpoint | Purpose |
|----------|---------|
| `/links` | Manage link data |
| `/analytics` | Access metrics |
| `/themes` | Customize appearance |

<ExpandableGroup>
  <Expandable title="Authentication" default-open="false">
    Use OAuth 2.0 or API keys for secure access.
  </Expandable>
  <Expandable title="Webhooks" default-open="false">
    Receive real-time notifications for link clicks.
  </Expandable>
</ExpandableGroup>

<CodeGroup tabs="Node.js,Ruby">
  ````javascript
  const axios = require('axios');
  
  const response = await axios.get('https://api.linktree.com/v1/integrations', {
    headers: { 'Authorization': `Bearer ${API_KEY}` }
  });
  console.log(response.data);
  ````
  ````ruby
  require 'net/http'
  require 'json'
  
  uri = URI('https://api.linktree.com/v1/integrations')
  http = Net::HTTP.new(uri.host, uri.port)
  request = Net::HTTP::Get.new(uri)
  request['Authorization'] = "Bearer #{API_KEY}"
  response = http.request(request)
  puts JSON.parse(response.body)
  ````
</CodeGroup>
