## API Troubleshooting

For developers using the API:

<Tabs>
  <Tab title="Authentication" icon="lock">
    Verify your API key is valid and has correct permissions.
  </Tab>
  <Tab title="Rate Limits" icon="clock">
    Check if you've exceeded API request limits.
  </Tab>
</Tabs>

| Error Code | Meaning | Solution |
|------------|---------|----------|
| 401 | Unauthorized | Check API key |
| 429 | Rate limited | Wait and retry |
| 500 | Server error | Contact support |

<CodeGroup tabs="Bash,PowerShell">
  ````bash
  # Test API connectivity
  curl -H "Authorization: Bearer YOUR_KEY" https://api.linktree.com/v1/status
  ````
  ````powershell
  # Test API connectivity
  Invoke-RestMethod -Uri 'https://api.linktree.com/v1/status' -Headers @{'Authorization' = 'Bearer YOUR_KEY'}
  ````
</CodeGroup>
